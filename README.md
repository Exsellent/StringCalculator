# ЗАДАЧА "СТРОКОВЫЙ КАЛЬКУЛЯТОР"

# Описание
Kонсольное приложение "Строковый калькулятор". Приложение читает из консоли введённые
пользователем строки, числа, арифметические операции проводимые между ними и выводить в консоль результат
их выполнения.

# Класс StringCalculator

Класс StringCalculator представляет собой консольное приложение, которое выполняет вычисления над
строками. Приложение принимает на вход строку, содержащую выражение, и выводит в консоль результат
вычисления.

# Метод main()

Метод main() является точкой входа в приложение. Он инициализирует объект класса Scanner для чтения
ввода с консоли, выводит приглашение пользователю ввести выражение и получает выражение от пользователя.
Затем метод вызывает метод calculate() для вычисления выражения и выводит результат вычисления в консоль.

# Метод calculate()

Выполняет вычисление выражения, заданного в качестве аргумента. Метод принимает на вход строку
содержащую выражение, и возвращает результат вычисления.
Метод сначала разделяет строку на три части с помощью регулярного выражения. Первая часть строки является
первым операндом, вторая часть - оператором, а третья часть - вторым операндом.
Затем метод проверяет, что выражение имеет правильную структуру. Если выражение имеет неправильную
структуру, то метод выбрасывает исключение IllegalArgumentException.
Далее метод выполняет вычисление выражения в соответствии с оператором. Для выполнения вычисления
используются методы addStrings(), subtractStrings(), multiplyStrings() и divideStrings().

После вычислений метод возвращает результат вычисления.

Методы для работы со строками

Класс StringCalculator содержит следующие методы для работы со строками:

    addStrings() - добавляет две строки.
    subtractStrings() - удаляет из первой строки вторую строку.
    multiplyStrings() - повторяет строку заданное количество раз.
    divideStrings() - возвращает строку, состоящую из первых n символов исходной строки.

Методы для проверки значений

Класс StringCalculator содержит следующие методы для проверки значений:

    isInteger() - проверяет, является ли строка целым числом.

Обработка исключений

Приложение обрабатывает следующие исключения:

    IllegalArgumentException - выбрасывается, если выражение имеет неправильную структуру или значения операндов не соответствуют требованиям.
    NumberFormatException - выбрасывается, если второй операнд не является целым числом.

Примеры использования приложения:

Введите выражение для вычисления:
"100" + "500"

Результат:
100500

Введите выражение для вычисления:
"Hi World!" - "World!"

Результат:
Hi

Введите выражение для вычисления:
"Bye-bye!" - "World!"

Результат:
Bye-bye

Введите выражение для вычисления:
"Java" * 5

Результат:
JavaJavaJavaJavaJava

"Example!!!" / 3

Результат:
Exa
